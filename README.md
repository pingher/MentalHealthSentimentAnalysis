Mental Health Sentiment Analysis

Overview
Mental health is an important aspect of overall well-being, and analyzing sentiment from text can provide valuable insights into mental health trends and individual emotions. 
This project aims to analyze sentiment related to mental health. The system is designed to classify text into various sentiment categories and provide insights into underlying emotional patterns.

To run
pip install -r requirements.txt
Launch JupyterLab and navigate to the project directory.
Open the terminal in JupyterLab and run the following command to start the app: streamlit run App2.py
